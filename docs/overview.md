# Linka Editor — Обзор задач по ТЗ

Версия ТЗ: 1.1 (31.07.2025)

---

## Что это за продукт

Веб-платформа для создания, редактирования и экспорта наборов карточек в формате `.linka` (ZIP-архив). Целевая аудитория — дефектологи, родители, педагоги, работающие с детьми с инвалидностью. Готовые наборы открываются в приложении Linka Looks.

---

## Что нужно сделать

### 1. Веб-редактор (Nuxt 3 + Vue 3)
- Авторизация: e-mail/пароль + Google SSO
- Создание и редактирование сетки карточек (columns × rows)
- Поиск и вставка изображений через Pictures Bank
- Загрузка собственных изображений и аудиофайлов
- Режим викторины (quiz, questions)
- История версий + откат
- Экспорт набора в файл `.linka` (скачать локально)
- PWA-оффлайн с синхронизацией при подключении

### 2. ИИ-ассистент
- Чат на естественном языке
- Генерация карточек по тексту (ответ — JSON-patch к config.json)
- TTS озвучка текста карточек через Yandex SpeechKit → mp3
- Автозаполнение полей `audioPath`, `audioText`, `audioVoice`
- Модель: OpenAI o3

### 3. Микросервисы (Python 3.12 / Go, FastAPI, gRPC)

| Сервис | Задача |
|---|---|
| **Auth-Service** | OAuth2, JWT (TTL 15 мин), RBAC (4 роли) |
| **API-Gateway** | TLS-терминация, маршрутизация, rate-limit |
| **Pack-Service** | CRUD наборов, черновики, версионирование, финализация → ZIP |
| **Media-Service** | Загрузка файлов, дедупликация, presigned URL |
| **AI-Service** | LLM + TTS, очереди через RabbitMQ |
| **Pictures-Bank** | REST API банка изображений (уже существует) |

### 4. Формат файла `.linka`
- ZIP-архив: `config.json` + медиафайлы
- `config.json` валидируется по JSON Schema 2020-12
- Лимит архива — 50 МиБ
- Поля: version, columns, rows, cards[], quiz, questions и др. (см. ТЗ §5.6)

### 5. Управление доступом
- 4 роли: `defectologist`, `parent`, `viewer`, `admin`
- Лимит хранилища: 10 ГиБ на организацию

### 6. Интеграция Pictures Bank (существующее API)
- Поиск (`GET /picture/search?query=`)
- Вставка с сохранением `pictureId` в метаданные карточки
- Авторизация через Bearer-токен
